get /beneficios
Obtener beneficios (beneficiosGet)
Este endpoint retorna los beneficios ordenados por años, filtrados por montos máximos y mínimos definidos en los filtros (endpoint), con información detallada de las fichas (endpoint) asociadas.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Listado de beneficios

Example data

Content-Type: application/json
{"code":200,"success":true,"data":[{"year":2023,"num":8,"beneficios":[{"id_programa":147,"monto":40656,"fecha_recepcion":"09/11/2023","fecha":"2023-11-09","ano":"2023","view":true,"ficha":{"id":922,"nombre":"Emprende","id_programa":147,"url":"emprende","categoria":"trabajo","descripcion":"Fondos ...."}}]}]}

500

Error interno del servidor
